# Tibber Add-on Repository

This is the Home Assistant add-on repository for **Tibber** data, allowing you to view your Tibber consumption data directly in Home Assistant.

## Overview

The Tibber add-on lets you:
- Log in with your Tibber account
- View the last year of consumption data from the Tibber API
- Display prices, consumption, and costs in a dashboard

## Installation

### Stable Channel

Add the repository to your Home Assistant add-on store:
`https://github.com/apps-homeassistant/Tibber`

### Beta Channel

For pre-release builds, use the beta branch:
`https://github.com/apps-homeassistant/Tibber#beta`

## 🚀 One-Click Installation (Stable)

Click the button below to add this repository to your Home Assistant instance:

[![Add Repository to Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fapps-homeassistant%2FTibber)

## 📖 How to Use

1. Add this repository URL to your Home Assistant Add-on Store.
2. Install the **Tibber** add-on.
3. Start the add-on and open the Web UI.
4. Log in with your Tibber credentials.

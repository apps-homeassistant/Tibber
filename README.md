# Tibber App Repository

This is the Home Assistant app repository for **Tibber** data, allowing you to view your Tibber consumption data directly in Home Assistant.

## Overview

The Tibber app lets you:
- Log in with your Tibber account
- View the last year of consumption data from the Tibber API
- Display prices, consumption, and costs in a dashboard
- Integrate with Home Assistant [MQTT server](https://my.home-assistant.io/redirect/integration/?domain=mqtt) to create sensors and automations based on your Tibber data 

## Installation

### Stable Channel

Add the repository to your Home Assistant app store:
`https://github.com/apps-homeassistant/Tibber`

### Beta Channel

For pre-release builds, use the beta branch:
`https://github.com/apps-homeassistant/Tibber#beta`

1. Add this repository URL to your Home Assistant App Store.
2. Install the **Tibber** app.
3. Start the app and open the Web UI.
4. Log in with your Tibber credentials.
